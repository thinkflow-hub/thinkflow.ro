# ContentFactory: AI Content Production System

A modular multi-factory AI content production system for scaling output.

## The Six Factories

1. **SEO Factory**: GEO-optimized articles with auto-publish.
2. **KDP Factory**: Puzzle books and workbooks with validation.
3. **Copy Factory**: Landing pages, email, ad copy.
4. **Fanvue Factory**: ComfyUI image generation pipeline.
5. **Clipping Factory**: Short-form video production.
6. **Fiverr Factory**: Automated service delivery.

## Core Architecture

Shared modules: LLM routing (OpenRouter + Ollama), quality control, financial ledger, queue management.

## Tech Stack

Python, FastAPI, Ollama, Qdrant, ComfyUI, yt-dlp, Whisper, MCP protocol.
